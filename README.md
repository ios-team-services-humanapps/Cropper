## Features
- Image cropping
- Image rotating
- Image straightening
- Image flipping
- Select and lock aspect ratio 
- Save and restore state, reset/re-edit

## Installation

QCropper is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'QCropper'
```
